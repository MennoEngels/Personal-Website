Discord Clone

For a personal project, I built a Discord clone using C# that allowed users to create accounts, send friend requests, block friends, chat with friends, see who's online, and watch back chat history. I developed this project during my time in school while learning how to use SQL in C#. I used it as an opportunity to practice my skills and apply what I was learning in class to a real-world project.

Building this project was a valuable experience that allowed me to deepen my understanding of SQL, C#, and software development in general. It also served as a testament to my ability to take on personal projects and develop fully functional applications from start to finish.
